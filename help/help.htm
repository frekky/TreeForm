TreeForm 0.2

Introduction

TreeForm 0.2 is a beta version of syntax tree drawing software intended to allow Syntacticians to draw complex trees using visual tools. This software allows the representation of Syntax trees with features such as theta roles, case markings, and feature sets. Because of the freedom of expression allowed through this tool, it is currently possible to express index and trace information, allowing this tool to be used for serious research. TreeForm 0.2 was written in Java, and will operate in Microsoft, Linux, and MacIntosh environments.

Setup

To install this software in your computer, you need to download the software from my website and unzip the folder in any folder of your choosing.

This software requires Java 1.4.2 or higher to run. The Java runtime can be downloaded from the Java Website. When at this page, download the runtime instead of the software development package (unless you ARE a software developer and want all the extra code).

To make the software work on your machine, run the batch program provided in the downloaded zip file named treeform.bat. This file starts the treeform program and makes sure that the computer requests enough memory to create jpeg and portable network graphics (png) files if requested. The program will still run if opened directly, but it will not let you save picture files of your trees.

To use Unicode fonts, you will need some kind of Unicode keyboard. In the windows environment, you can use Tavultesoft Keyman found at: Tavultesoft. As of the writing of this document, the current version of Keyman is Keyman 6.0.164.0.

The most notable Unicode font for linguists is the Doulous SIL font. This font contains all the IPA symbols and diacritics, and is invaluable to any linguist wishing to represent IPA codes in a font usable by all Unicode computer systems. The font can be found at: The Doulos SIL site. Download the font appropriate to your system.

To use the font in Tavultesoft Keyman, you will need the proper keyboard. This keyboard can be found at: the SIL site. I recommend that you then unzip the file, and run Keyman. Then right-click the keyman icon in your windows taskbar, and select the menu option "Keyman Configuration". Then select the "Install keyboard" option, and install the new keyboard. You will be able to select that keyboard in your program, and type with it. (This keyboard comes with a PDF file detailing how to use it.)

Reordering a Tree

If you move your mouse over a part of your tree and right-click (apple-click on MACs) you will be given the option to delete or reposition subtrees.



Deleting a subtree permanently delete the highlighted tree item and all of it’s subtrees. Repositioning a subtree has the same effect as adding a new structure to a binary (or larger) tree element, you get a screen that looks like this:



Move the mouse until the green dot is in the position in which you want the new tree drawn. Then left-click and the tree will be drawn appropriately.





Changing Text

Changing text is easy. All you need to do is move your mouse over the text you want to change, and left-click. Then move the cursor position as you would in any drawing program, type, delete, use backspace, copy and paste as you would in any text editor program.

Adding Associations

To add an association, hold down shift and left-click the feature you want to associate:



then hold the mouse button down and drag the feature until you highlight the associated feature. You can then release the mouse button, and the association will be complete.



Changing text in a feature changes the text in all the associations.
Note that by right-clicking an association, you can get a menu option to delete the association.

TreeForm

Setup

To install this software in your computer, unzip the folder in any folder of your choosing.

This software requires Java 1.4.2 or higher to run. The Java runtime can be downloaded from the Java Website.

This program requires DOULOS SIL fonts to look it's best. Download and install them from The Doulos SIL site

For best results on a PC, run the program from 'treeform.bat', on a mac, run the program from 'TreeForm.app'

Commands

To build a new tree - Click on any of: Head, Phrase, Adjunct, Unary, Binary, Trinary, or X-bar.

To add to a tree - Click and DRAG any of the above, or morph or triangle, until you have highlighted another part of the tree in Blue. It will either appear, or if you are adding a sister node, you will get red and green positioning dots. When you click again, the tree will insert into the place of the large green dot.

To move a tree or part of a tree - SHIFT click on a node, and drag the node onto another node (which will highlight blue).

To add a movement line - ALT click on a node, and drag the node onto another node (which will highlight in blue).

To add a feature set or case marking - Click and DRAG them onto a node.

To add an association (like associated theta roles) - SHIFT Click and DRAG them onto another node.

To delete a part of a tree, feature, feature set, association or movement line - RIGHT click on a node, and select relevant popup menu option.

To change any text - CLICK on a node, and type, highlight, copy, delete and paste as you would in any word processor. This works for font changes, font sizes, and font effects.

To undo any changes - CLICK on the UNDO or REDO buttons.

To make a tree bigger or smaller - CLICK the ZOOM IN or ZOOM OUT button. ZOOM effects printouts, what you see is literally what you get.

To save trees for later editing - SAVE and LOAD trees as you would any word processing document - through the menu or icons.

To paste a tree into MS Word - CLICK on the copy tree icon or menu option, and PASTE the tree into MS word as you would any other clipboard item. You will get a 300dpi image which should be RESIZED at 24% (new versions of MS Word do this automatically because they read my headers properly)

To use a tree in LaTeX - PRINT the tree as a PDF, and insert the tree as a figure into your document. You will get a beautiful vector graphic of the tree.

To use a tree on the Internet - EXPORT the tree as a PNG of JPG of desired screen resolution.
